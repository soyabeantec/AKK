validPeriodRange

(JVM) private fun validPeriodRange(checkPeriod: Int): Boolean

Checks if a Int is a period range. To be valid is has to consist of only six characters. The first four digits have to be a valid year between '0000' and '9999'. The last two digits have a be a valid month between '01' and '12'.

Parameters

checkPeriod - the to be checked company code.

Return
'true' when it fulfills the given requirements.
