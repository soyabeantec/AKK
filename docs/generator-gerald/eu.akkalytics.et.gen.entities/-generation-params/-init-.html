(JVM) GenerationParams(company: String, plant: String, importCountry: String, periodRangeStart: Int, periodRangeEnd: Int)

A data class which holds the parameters that are passed at the start of the program and are later utilized as a basis for the customs data generation.

Parameters

company - a two letter code, e.g. ‘WV’.

plant - a two digit numeric code, e.g. ’01’.

importCountry - a country code, e.g. ‘DE’.

periodRangeStart - a date (date format: yyyyMM), e.g. '201904'. Must be smaller than periodRangeEnd.

periodRangeEnd - a date (date format: yyyyMM)e.g. '201905'. Must be bigger than periodRangeStart.
