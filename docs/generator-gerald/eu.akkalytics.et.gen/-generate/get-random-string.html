getRandomString

(JVM) private fun Int.getRandomString(): String

Int extension function that: Generates a random string with the length according to the Int value. Uses only the ALLOWED_CHARACTERS for the generation. Example: 2.getRandomString() will generate something like this: "BK"

Return
a random string with the provided length.
