genDeclarationLastModified

(JVM) fun genDeclarationLastModified(): Date

Generates a day between the last invoice modification or declaration date, depending on which one is later and the actual date, if the shipment has already been sent, the shipment date will be used instead of the actual date.

Return
the day of the last modification as a Date obj.
